# Fantasy Globe Explorer

Files included:
- `index.html`
- `style.css`
- `app.js`

## Run locally

Because the page uses CDN scripts, easiest is to serve it with a tiny local server:

```bash
cd one-piece-globe
python3 -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

## What this starter includes

- Rotating 3D globe
- Clickable markers
- Route arcs between locations
- Info card popup
- Toggle labels
- Toggle routes
- Pause/resume auto-spin
- Reset camera
